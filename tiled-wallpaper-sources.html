tiled wallpaper sources

Tiled wallpapers are quirky, fun, and smol. That last property means you can do fun things to them fast, like programattically apply image filters or tint them so some current colorscheme.

The following is mostly a scraping of this nixer’s thread, and also the #rice wiki wallpapers (a tiled subset). The archives are compressed collections hosted on teknik.io. This collection focuses only on tiled patterns.

  • colour lovers patterns
  • hero patterns
  • stripe generator
  • propaganda tiles
  • the pattern library
  • dkeg’s bitmaps
  • wallace’s tiles and such
  • subtle patterns
  • http://cs.gettysburg.edu/~duncjo01/archive/patterns/
  • 1-background
  • tartan maker
  • http://bgrepeat.com archive
  • http://squidfingers.com/patterns archive
  • vain’s archive (eclipses tiles-and-such(?) and squidfingers)

🪨 🪨 🪨

For non-tiled sources, I usually go to unsplash or wallhaven.
